# Thư mục Logo

## Yêu cầu

- **Tên file:** `logo.png` (hoặc cập nhật trong `pubspec.yaml`)
- **Định dạng:** PNG (khuyến nghị) hoặc JPG
- **Kích thước:** Tối thiểu 1024x1024 pixels (càng lớn càng tốt)
- **Tỷ lệ:** Hình vuông (1:1)
- **Nền:** Trong suốt hoặc nền màu đẹp

## Cách sử dụng

1. Đặt file `logo.png` vào thư mục này
2. Chạy: `dart run flutter_launcher_icons`
3. Icon sẽ được tự động tạo cho tất cả nền tảng

## Ví dụ

```
assets/
└── logo/
    └── logo.png  ← File icon của bạn ở đây
```
